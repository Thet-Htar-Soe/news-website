<h5>How to run the project</h5>
<ul>
    <li>
        Clone the project
    </li>
    <li>
        Go to the folder application using cd command on your cmd or terminal
    </li>
    <li>
        Run npm install on your cmd or terminal
    </li>
    <li>
        Run npm start
    </li>
    <li>
        Go to http://localhost:3000/
    </li>
</ul>
